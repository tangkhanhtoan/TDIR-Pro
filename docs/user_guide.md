## 3. 🚀 Main Features / Tính năng chính
3.1 🧩 Multi-Panel
You can open multiple panels at the same time and change layout via Layout menu:
Row (Vertical), Column (Horizontal), Grid (2 columns).
Bạn có thể mở nhiều panel cùng lúc và thay đổi bố cục qua menu Layout: Row (Dọc), Column (Ngang), Grid (Lưới).
3.2 🔄 Undo (Ctrl + Z)
Supports undo for the following operations:
Delete, Copy, Cut, Rename, Duplicate, Extract, New Folder.
Hỗ trợ hoàn tác cho các thao tác: Xóa, Copy, Cut, Đổi tên, Nhân bản, Giải nén, Tạo thư mục mới.
3.3 📦 Zip & Extract
Select files/folders → Right-click → Create Zip File or Extract Here.
Chọn file/thư mục → Chuột phải → Create Zip File hoặc Extract Here.
3.4 🔍 Search
Press 🔍 button or Ctrl + F.
Supports wildcard (*.pdf, report*) and regex (start with re:).
Nhấn nút 🔍 hoặc Ctrl + F. Hỗ trợ tìm kiếm Wildcard và Regex.
3.5 🎨 Interface
Switch between Light and Dark theme
Adjust font size via View → Font Size
Chuyển đổi giao diện Light / Dark và thay đổi kích thước chữ qua View → Font Size.
